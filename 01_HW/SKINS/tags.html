Deprecated/Rarely Used Tags (For Reference)

Legacy Elements

The following are older tags, some deprecated but still recognized:

  • <center> - Centered text (deprecated, use CSS)
  • <font> - Font styling (deprecated, use CSS)
  • <strike> - Strikethrough (use <s> or <del>)
  • <big> - Bigger text (deprecated, use CSS)
  • <tt> - Teletype text (use <code> or CSS)
  • <frame>, <frameset> - Frames (obsolete)
  • <acronym> - Use <abbr> instead
  • <applet> - Java applets (obsolete)
  • <basefont> - Base font (deprecated)
  • <dir> - Directory list (use <ul>)
  • <marquee> - Scrolling text (non-standard)
  • <blink> - Blinking text (obsolete)
